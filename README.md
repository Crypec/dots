# Dotfiles

![desktop]("./readme_img.png")

## How to use it

1. Clone this repo:

    ```shell
    git clone --recursive https://github.com/crypec/dots.git $HOME/.dots
    cd $HOME/.dots
    ```
2. Install GNU Stow:

    ```shell
    # On Arch Linux
    sudo pacman -S stow
    ```

## Updating
To update the dotfiles pull this repo including its submodules.

```
git pull --recurse-submodules
```
